To Be, or :Not() to Be

<style>

#big-box{ background-color: gray; }

.small-box{ background-color: lightskyblue; }

.small-box.left { float: left; }
.small-box.right { float: right; }
					
.small-box:not(.left) { background-color: hotpink; }

</style>
